# subway_surfers
Бесконечный раннер, где персонаж бежит по городским путям, уворачивается от препятствий, собирает монеты и бонусы. Игрок может прыгать, скользить и менять рельсы, чтобы избегать столкновений. Цель — набрать как можно больше очков и собрать редкие предметы.
